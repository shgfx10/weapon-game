# Weapon Selection Game

A simple beginner-friendly Python game where the user must choose between two weapons: **Fire Sword** and **Ice Sword**.  
Made by **me**.

## How It Works
- The player is asked to choose a weapon.
- If they choose an invalid weapon, the game keeps asking.
- Once a valid weapon is chosen, the game reveals its special power.

## Features
- Input validation
- Loop control
- Beginner-friendly Python code
- Clean and easy to understand
